# 📿 Compteur de Dhikr

Une application web minimaliste, élégante pour accompagner vos invocations quotidiennes.

![License](https://img.shields.io/badge/license-MIT-green)
![Status](https://img.shields.io/badge/Status-Stable-blue)

## ✨ Fonctionnalités
- 🎯 **Objectifs personnalisables** : Fixez votre propre but quotidien.
- 🔥 **Système de Série (Streak)** : Suivez votre régularité jour après jour.
- 🔊 **Retour sensoriel** : Vibration et son discret à chaque clic (optimisé pour iPhone & Android).
- ✨ **Récompense visuelle** : Message de félicitations une fois l'objectif atteint.
- 💾 **Sauvegarde locale** : Vos données restent sur votre appareil (vie privée respectée).
- 📱 **Mobile First** : Interface conçue pour une utilisation à une main.


## 🔒 Sécurité & Confidentialité
Ce projet n'utilise aucun cookie tiers et ne collecte aucune donnée personnelle. Tout est géré via le `localStorage` du navigateur.
